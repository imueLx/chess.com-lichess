# Chess.com → Lichess

A simple Chrome extension that lets you analyze your Chess.com games on Lichess with one click.

---

## Install

1. **Download** this repo — click the green **Code** button → **Download ZIP**, then extract it.
2. Open **Chrome** and go to `chrome://extensions`.
3. Turn on **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** and select the extracted folder.
5. Pin the extension icon in your toolbar for easy access.

## How to Use

1. Go to any game on **Chess.com** (e.g. `chess.com/game/live/123456789`).
2. Click the **Chess.com → Lichess** icon in your toolbar.
3. Hit **Analyze on Lichess** — a Lichess tab opens with the PGN already filled in.
4. Click **Import** on Lichess and you're done.

You can also use the **floating button** that shows up in the bottom-right corner of any Chess.com game page, or use the **Copy PGN** button to grab the PGN to your clipboard.
