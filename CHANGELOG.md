# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-03-07

### Added

- One-click PGN extraction from Chess.com game pages
- Automatic PGN paste into Lichess import page
- Floating action button (FAB) on Chess.com game pages
- Copy PGN to clipboard button
- Support for live, daily, and archived games
- 7 extraction strategies (API + DOM fallback)
- Figurine icon handling (`data-figurine` attribute support)
- Board flip detection for correct player name assignment
- Dark-themed popup UI
- Privacy policy
